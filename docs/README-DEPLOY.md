# 🚀 Deploy Rápido

## Comandos Disponíveis

### Deploy Completo (Recomendado)

```bash
npm run deploy
```

✅ Faz build + upload da pasta `dist` para o servidor

### Deploy Rápido

```bash
npm run deploy:only
```

⚡ Apenas upload (sem build)

## Acesso ao Site

Após o deploy, acesse:

-   🌐 http://ram4org1.infinityfree.com
-   🌐 http://dronequest.free.nf
